# Backend Technical Test

## Overview

This project is a backend API built with **Express.js** as part of a technical assessment. The application demonstrates RESTful API development, database operations, scheduled background jobs (cron jobs), input validation, and structured error handling following backend development best practices.

## Features

* RESTful API using Express.js
* Database integration
* Scheduled tasks using cron jobs
* Request validation
* Centralized error handling

## Tech Stack

* **Runtime:** Node.js
* **Framework:** Express.js
* **Database:** PostgreSQL / MySQL / MongoDB *(depending on your configuration)*
* **Package Manager:** npm

## Prerequisites

Before running the project, ensure you have the following installed:

* Node.js (v18 or newer recommended)
* npm
* Git
* A supported database server

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/rivando-al-rasyid/huawei-technical.git
cd huawei-technical
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the application

Development mode:

```bash
npm run dev
```

Production mode:

```bash
npm start
```

The API will be available at:

```text
http://localhost:3000
```

## Running Tests

Run all tests:

```bash
npm test
```
